# Grounded Frontend Spec

## Objective
Minimal, visually striking frontend for NL → Query → SQL → Answer system.

## Core Features
- Query input
- Answer display
- Results table
- QueryPlan + SQL + Trace panels
- Auth gate (feature-flagged)

## Layout
Top: title + input  
Bottom: two columns (results | internals)

## States
- Loading
- Failure
- Locked (auth)

## Design
- Dark, premium
- Minimal, no clutter
